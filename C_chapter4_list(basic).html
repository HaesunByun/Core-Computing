리스트(기초)#

학습목표

파이썬은 다수의 데이터를 하나로 묶어서 처리할 수 있는 다양한 자료형을 제공한다. 그 중에 하나인 리스트를 알아보자.

이 장에서는 리스트의 개념, 리스트의 인덱싱, 슬라이싱, 리스트 연산자 등에 대해서 알아본다.

리스트 생성#

  • 리스트는 숫자, 문자, 리스트, 딕셔너리 등 어떠한 데이터 타입이든 다 담을 수 있는 만물박스이다.

  • 리스트는 대괄호[ ]로 묶어준다.

  • 리스트에 들어있는 데이터를 아이템, 원소, 요소이라고 부른다.

  • 리스트 내 아이템은 컴마(,)로 구분한다.

직접 생성#

  • list() 함수는 빈 리스트, 숫자리스트, 문자열을 낱개로 분리하여 리스트로 만드는 함수이다.

a = [1, 2, 3, 4, 5]
b = [3, 8, 1, 3, 2]
c = [1, 'apple', ['banana',1, 2], 'coconut']
d = ['apple', 'banana', 'coconut', 'durian']
e = [ ]

list() 함수로 생성#

  • list() 함수는 빈 리스트, 숫자리스트, 문자열을 낱개로 분리하여 리스트로 만드는 함수이다.

item = list()

a=list('hello')
print(a)

['h', 'e', 'l', 'l', 'o']

a=list(range(1,10))
print(a)

[1, 2, 3, 4, 5, 6, 7, 8, 9]

split() 메서드로 생성#

  • split() 함수는 문자열을 특정 기준에 따라 분리하여 리스트로 만드는 함수이다. 형식은 다음과 같다.

문자열.split()

  • split(문자열) 괄호안의 문자열을 기준으로 분리하며, 괄호안에 아무것도 없으면 공백을 기준으로 분리한다.

item = '바나나 딸기 복숭아'
fruit = item.split()
print(fruit)

['바나나', '딸기', '복숭아']

item = '바나나, 딸기, 복숭아'
fruit = item.split(', ')
print(fruit)

['바나나', '딸기', '복숭아']

인덱싱(Indexing)#

  • 인덱스는 리스트 내 아이템의 주소이다.

양의 인덱스(positive index)#

positive index는 가장 왼쪽 아이템에서 0부터 시작하여 오른쪽으로 증가한다.

shape= ['turtle', 'arrow', 'circle', 'square']
shape[0]

'turtle'

  • 리스트안에 리스트가 있는 경우 대괄호를 연속하여 쓰면 된다.

  • 예를 들어, ‘banana’를 꺼내려면 c[2][0]과 같이 접근할 수 있다.

c=[1, 'apple', ['banana',1, 2], 'coconut']
print(c[2])
print(c[2][0])

['banana', 1, 2]
banana

음의 인덱스(negative index)#

  • negative index는 가장 오른쪽 끝 아이템에서 -1부터 시작하여 왼쪽으로 감소한다.

d=['apple', 'banana', 'coconut', 'durian']
print(d[-1], d[-2], d[-3], d[-4])

durian coconut banana apple

# list index out of range
print(d[4])
print(d[-5])

슬라이싱(Slicing)#

슬라이싱이란 리스트의 아이템 일부를 자르는 것이다. 형식은 다음과 같다.

리스트변수[start:end:step]

  • start: 시작 인덱스

  • end: 끝 인덱스+1

  • step: 이동간격, 양수이면 오른쪽으로 step만큼 이동한다. 음수이면 왼쪽으로 step만큼 이동한다.

  • start, end, step 모두 생략 가능하다.

  • 아래의 코드를 통해 실습해보자. 처음부터 인덱스 3 이전까지 왼쪽으로 이동하여 슬라이싱한다.

shape= ['turtle', 'arrow', 'circle', 'square', 'diamond']
print(shape[ :3])

  • index -3부터 끝까지 왼쪽으로 이동하여 슬라이싱한다.

print(shape[-3: ])

  • 처음부터 -3 이전까지 왼쪽으로 이동하여 슬라이싱한다.

print(shape[:-3])

  • 처음부터 -3이전까지 오른쪽으로 이동하여 슬라이싱한다.

print(shape[:-3:-1])

리스트 주요 메서드#

사용 방법

의미

예시(**a=[‘a’,’b’,’c’]**일 때)

list.append(item)

list의 마지막에 item을 추가

a.append(‘d’) ➡ [‘a’,’b’,’c’,’d’]

list.extend(list2)

list의 마지막에 list2의 아이템을 추가하여 확장

a.extend([‘d’,’e’]) ➡ [‘a’,’b’,’c’,’d’,’e’]

list.insert(index, item)

list의 index 위치에 item을 삽입

a.insert(1,’d’) ➡ [‘a’,’d’,’c’,’d’]

list.remove(item)

list에서 첫 번째로 나오는 item을 삭제

a.remove(‘c’) ➡ [‘a’,’c’]

list.index(item)

list에서 item의 index를 반환

a.index(‘c’) ➡ 2

list.count(item)

list에서 item의 개수를 반환

a.count(‘c’) ➡ 1

list.clear()

list의 모든 item의 제거한다.

a.clear() ➡ []

list.pop(index)

list에서 인덱스의 아이템을 제거하고 반환.
인덱스를 지정하지 않으면 마지막 아이템을 제거하고 반환

a.pop(1) ➡ [‘a’,’c’]

list.reverse()

list에서 아이템의 순서를 뒤집는다.

a.reverse() ➡ [‘c’,’b’,’a’]

list.copy()

list를 shallow copy한다.

b=a.copy() ➡ b= [‘a’,’b’,’c’]

list.sort()

list 내 item를 오름차순으로 정렬

a = [‘c’,’a’,’b’]**일 때, a.sort() ➡ [‘a’, ‘b’, ‘c’]

list2 = sorted(list)

list를 오름차순으로 정렬하여 반환

b = sorted(a) ➡ b = [‘a’, ‘b’, ‘c’]
