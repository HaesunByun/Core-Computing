리스트(기초)#

학습목표

파이썬에서는 다수의 데이터를 하나로 묶어서 처리할 수 있는 다양한 자료형을 제공한다. 그 중에 하나인 리스트를 알아보자.

이 장에서는 리스트의 개념, 리스트의 인덱싱, 슬라이싱, 리스트 연산자 등에 대해서 알아본다. :::

리스트 생성#

  • 리스트는 숫자, 문자, 리스트, 딕셔너리 등 어떠한 데이터 타입이든 다 담을 수 있는 만물박스이다.

  • 리스트는 대괄호[ ]로 묶어준다.

  • 리스트에 들어있는 데이터를 아이템, 원소, 요소이라고 부른다.

  • 리스트 내 아이템은 콤마(,)로 구분한다.

  • 형식은 다음과 같다.

[아이템1, 아이템2, 아이템3, ..., 아이템n]

직접 생성#

  • 대괄호 안에 아이템을 콤마로 나열하여 직접 생성할 수 있다.

  • 아이템없이 대괄호만 있으면 비어있는 리스트를 생성한다.

a = [1, 2, 3, 4, 5]
b = [3, 8, 1, 3, 2]
c = [1, 'apple', ['banana',1, 2], 'coconut']
d = ['apple', 'banana', 'coconut', 'durian']
e = [ ]

list() 함수로 생성#

  • list() 함수는 빈 리스트, 숫자리스트, 문자열을 낱개문자로 분리하여 리스트로 만드는 함수이다.

item = list()

  • range()함수와 같이 사용하면 수열로 이루어진 리스트를 생성한다.

a=list(range(1,10))
print(a)

[1, 2, 3, 4, 5, 6, 7, 8, 9]

range() 함수

사용 방법

의미

예시

range(끝)

[0, 1, 2, 3, …, 끝-1]

range(4) → [0, 1, 2, 3]

range(시작, 끝)

[시작, 시작+1, 시작+2, …, 끝-1]

range(3, 5) → [3, 4]

range(시작, 끝, 증감)

[시작, 시작+증감, 시작+증감×2, …, 시작+증감×k]

range(2, 11, 2) → [2, 4, 6, 8, 10]

range(시작, 끝, -증감)

[시작, 시작-증감, 시작-증감×2, …, 시작-증감×k]

range(9, 1, -2) → [9, 7, 5, 3]

  • range() 함수는 일정한 규칙에 따라 나열된 수열을 생성한다.

  • 끝은 범위에 미포함이다.

  • range 함수의 결과는 리스트 형태로 나오지 않는다. 리스트로 보고싶으면 list() 함수와 같이 사용한다.

  • 문자열을 각 문자로 분리하여 리스트를 생성한다.

split() 메서드로 생성#

  • split() 함수는 문자열을 특정 기준에 따라 분리하여 리스트로 만드는 함수이다.

  • 문자열을 괄호안의 ‘분리기준’으로 분리하며, 괄호안에 분리기준이 없으면 공백을 기준으로 분리한다.

  • 분리되면서 분리기준은 제거된다.

사용 방법

의미

예시

split()

공백(’ ‘)을 기준으로 문자열을 나눠 리스트를 생성

‘1 2 3 4 5’.split() → [‘1’, ‘2’, ‘3’, ‘4’, ‘5’]

split(분리기준)

분리기준를 기준으로 문자열을 나눠 리스트를 생성

‘1:2:3:4:5’.split(‘:’) → [‘1’, ‘2’, ‘3’, ‘4’, ‘5’]

  • split() 괄호안에 분리기준이 없으므로 공백을 기준으로 분리한다.

item = '바나나 딸기 복숭아'
fruit = item.split()
print(fruit)

['바나나', '딸기', '복숭아']

  • split()괄호안에 적히 ‘, ‘를 기준으로 분리한다.

  • 이때 콤마만 기준으로 주지 않고 콤마 다음에 공백까지 포함하여 분리하면 공백까지 제거된다.

item = '바나나, 딸기, 복숭아'
fruit = item.split(', ')
print(fruit)

['바나나', '딸기', '복숭아']

인덱싱(Indexing)#

  • 인덱스는 리스트 내 아이템의 주소이다. 즉, 아이템에 접근하기 위해 인덱스를 사용한다.

양의 인덱스(positive index)#

positive index는 가장 왼쪽 아이템에서 0부터 시작하여 오른쪽으로 1씩 증가한다.

shape= ['turtle', 'arrow', 'circle', 'square']
print(shape[0])
print(shape[1])
print(shape[2])
print(shape[3])

turtle
arrow
circle
square

  • 리스트안에 리스트가 있는 경우 대괄호를 연속하여 쓴다.

  • 예를 들어, ‘banana’는 c[2][0]과 같이 접근할 수 있다.

c=[1, 'apple', ['banana',1, 2], 'coconut']
print(c[2])
print(c[2][0])

['banana', 1, 2]
banana

음의 인덱스(negative index)#

  • negative index는 가장 오른쪽 끝 아이템에서 -1부터 시작하여 왼쪽으로 1씩 감소한다.

d=['apple', 'banana', 'coconut', 'durian']
print(d[-1], d[-2], d[-3], d[-4])

durian coconut banana apple

  • 없는 인덱스에 접근하면 에러가 난다.

print(d[4])
print(d[-5])

아이템 수정#

  • 리스트안의 아이템은 인덱스를 이용하여 수정한다.

리스트변수명[인덱스]=값

  • 리스트변수 shape의 인덱스2의 값을 100으로 수정한다.

shape= ['turtle', 'arrow', 'circle', 'square', 'diamond']
shape[2]=100
print(shape)

['turtle', 'arrow', 100, 'square', 'diamond']

슬라이싱(Slicing)#

슬라이싱이란 리스트의 아이템 일부를 자르는 것이다. 형식은 다음과 같다.

리스트변수[start:end:step]

  • start: 슬라이싱 시작 인덱스

  • end: 슬라이싱 끝 인덱스(끝 인덱스는 미포함)

  • step: 이동간격, 양수이면 오른쪽으로 step만큼 이동한다. 음수이면 왼쪽으로 step만큼 이동한다.

  • start, end, step 모두 생략 가능하다.

  • 아래의 코드를 통해 실습해보자. 처음부터 인덱스 3 이전까지 오른쪽으로 1씩 이동하여 슬라이싱한다.

shape= ['turtle', 'arrow', 'circle', 'square', 'diamond']
print(shape[ :3])

['turtle', 'arrow', 'circle']

  • 인덱스 -3부터 끝까지 오른쪽으로 1씩 이동하여 슬라이싱한다.

print(shape[-3: ]) 

['circle', 'square', 'diamond']

  • 처음부터 인덱스 -3 이전까지 오른쪽으로 1씩 이동하여 슬라이싱한다.

print(shape[:-3])

['turtle', 'arrow']

  • 처음부터 인덱스 -3 이전까지 왼쪽으로 1씩 이동하여 슬라이싱한다.

print(shape[:-3:-1])

['diamond', 'square']

  • 처음부터 인덱스 -3 이전까지 왼쪽으로 2씩 건너뛰면서 슬라이싱한다.

print(shape[:-3:-2])

['diamond']

리스트 주요 메서드#

사용 방법

의미

예시(a=[‘a’,’b’,’c’]일 때)

list.append(item)

list의 마지막에 item을 추가

a.append(‘d’) ➡ [‘a’,’b’,’c’,’d’]

list.extend(list2)

list의 마지막에 list2의 아이템을 추가하여 확장

a.extend([‘d’,’e’]) ➡ [‘a’,’b’,’c’,’d’,’e’]

list.insert(index, item)

list의 index 위치에 item을 삽입

a.insert(1,’d’) ➡ [‘a’,’d’,’c’,’d’]

list.remove(item)

list에서 첫 번째로 나오는 item을 삭제

a.remove(‘c’) ➡ [‘a’,’c’]

list.index(item)

list에서 item의 index를 반환

a.index(‘c’) ➡ 2

list.count(item)

list에서 item의 개수를 반환

a.count(‘c’) ➡ 1

list.clear()

list의 모든 item의 제거한다.

a.clear() ➡ []

list.pop(index)

list에서 인덱스의 아이템을 제거하고 반환.
인덱스를 지정하지 않으면 마지막 아이템을 제거하고 반환

a.pop(1) ➡ [‘a’,’c’]

list.reverse()

list에서 아이템의 순서를 뒤집는다.

a.reverse() ➡ [‘c’,’b’,’a’]

list.copy()

list를 shallow copy한다.

b=a.copy() ➡ b= [‘a’,’b’,’c’]

list.sort()

list 내 item를 오름차순으로 정렬

a = [‘c’,’a’,’b’]일 때, a.sort() ➡ [‘a’, ‘b’, ‘c’]

list2 = sorted(list)

list를 오름차순으로 정렬하여 반환

b = sorted(a) ➡ b = [‘a’, ‘b’, ‘c’]

List comprehension(리스트 축약법)#

  • 리스트를 생성할 때 더 간결한 방법으로 생성하는 문법이다.

  • 대괄호안에 for문이나 if문을 이용해 한 줄의 코드로 리스트를 만든다.

  • 리스트 축약뿐만 아니라 딕셔너리 축약, 집합축약도 만들 수 있다.

  • 이러한 축약법은 파이썬이 파이썬스럽다 라는 것을 보여주는 대표적인 예이다.

  • 아래 예를 통해 풀어 쓴 코드와 축약형 코드의 차이를 확인해보자.

print('풀어 쓴 코드======')
x=[ ]
for i  in  range(1,11): 
    x.append(i**2)
print(x)

print('축약형 코드======')
x=[i**2 for i  in  range(1,11)]
print(x)

풀어 쓴 코드======
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
축약형 코드======
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

마무리#

  • list는 다수의 데이터를 묶어 하나의 변수로 처리할 때 유용하다.

  • 리스트의 인덱스는 0부터 시작한다.

  • list( ) 함수를 활용하여 문자열이나 숫자 리스트를 생성한다.

  • split( ) 함수는 문자열을 분리하여 리스트를 생성한다.

  • range( )함수는 규칙적인 숫자열로 구성된 리스트를 생성한다.

  • 리스트에 아이템 추가는 객체명.append( )를 활용한다.

  • 리스트에 리스트 추가는 객체명.extend( )를 활용한다.

  • 리스트에 아이템을 삽입은 객체명.insert( )를 활용한다.

  • 리스트에서 첫번째로 나오는 아이템 제거는 객체명.remove( )를 활용한다.

  • 아이템의 인덱스 번호를 검색할 때에는 객체명.index( )를 활용한다.

  • 리스트의 아이템 정렬은 sort()와 sorted()를 활용한다.
